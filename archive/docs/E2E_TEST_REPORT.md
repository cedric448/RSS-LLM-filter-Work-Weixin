### ⚠️ 已知问题

1. **Mock RSS Server 不稳定**
   - 状态: 线程资源耗尽,重启循环
   - 影响: 无法获取真实 RSS 数据
   - 解决方案: 
     - 短期: 关闭 debug 模式（已完成）
     - 长期: 使用 gunicorn 替代 Flask 开发服务器
   - 优先级: P1（高）

2. **端口配置非标准**
   - HTTPS 使用 8443 而非 443
   - 原因: 443 端口被其他服务占用
   - 影响: 用户需要记住端口号
   - 解决方案: 
     - 使用域名 + DNS 配置
     - 或调整现有服务端口分配
   - 优先级: P3（低）
