## 快速开始

### 系统概览

RSS智能筛选推送系统会自动完成以下工作：

1. **定时获取**: 每天在 9:00、13:00、20:00 自动运行
2. **内容获取**: 从配置的公众号RSS源获取最新文章
3. **智能筛选**: 使用AI大模型判断文章是否与AI主题相关
4. **去重处理**: 自动过滤已推送的文章，避免重复
5. **消息推送**: 将筛选后的文章推送到企业微信群

## 调整筛选逻辑

### 理解筛选机制

系统使用**语义匹配**而非关键词精确匹配：
- ✅ 理解文章主旨和上下文
- ✅ 支持同义词和相关概念
- ❌ 不是简单的关键词搜索

**示例**:
- "OpenAI发布新模型" → 匹配 "大模型" 分类 ✅
- "某AI公司完成融资" → 匹配 "AI融资" 分类 ✅
- "餐厅使用AI点餐" → 不匹配（非核心AI资讯）❌


#### Cron 表达式说明

格式: `分钟 小时 日 月 星期 命令`

| 字段 | 取值范围 | 说明 |
|------|---------|------|
| 分钟 | 0-59 | - |
| 小时 | 0-23 | 0为午夜，12为正午 |
| 日 | 1-31 | - |
| 月 | 1-12 | - |
| 星期 | 0-7 | 0和7都表示周日，1-6表示周一到周六 |

**特殊字符**:
- `*`: 所有值
- `,`: 列举多个值，如 `9,13,20`
- `-`: 范围，如 `1-5` (周一到周五)
- `/`: 间隔，如 `*/3` (每3小时)

### 调整推送内容格式

#### 修改消息模板

编辑 `src/wechat_pusher.py`:

```python
def format_message(articles, total_count):
    """
    自定义消息格式
    """
    now = datetime.now()
    timestamp = now.strftime('%Y-%m-%d %H:%M')
    
    # ===== 可修改部分开始 =====
    
    # 消息头部
    message = f"## 📰 AI资讯精选 ({timestamp})\n\n"
    message += f"本期筛选 {total_count} 篇文章，推送 {len(articles)} 篇\n\n"
    message += "---\n\n"
    
    # 文章列表格式
    for i, article in enumerate(articles, 1):
        # 标题
        message += f"### {i}. {article['title']}\n"
        
        # 分类标签
        categories = ' | '.join(article['categories'])
        message += f"**分类**: {categories}  \n"
        
        # 来源
        message += f"**来源**: {article['source']}  \n"
        
        # 摘要（可选显示）
        if article.get('summary'):
            summary = article['summary'][:100] + '...'
            message += f"**摘要**: {summary}  \n"
        
        # 原文链接
        message += f"[阅读原文]({article['url']})\n\n"
        message += "---\n\n"
    
    # 消息尾部
    message += "*Powered by RSS智能筛选系统*"
    
    # ===== 可修改部分结束 =====
    
    return message
```

**自定义选项**:

1. **修改表情符号**:
   - 📰 → 📱 或 🔥
   - 更换分类图标

2. **调整摘要长度**:
   ```python
   summary = article['summary'][:150] + '...'  # 改为150字
   ```

3. **添加推荐理由**:
   ```python
   if article.get('reason'):
       message += f"**推荐**: {article['reason']}  \n"
   ```

4. **调整分隔线样式**:
   ```python
   message += "━━━━━━━━━━━━━━━\n\n"  # 更明显的分隔
   ```

### 临时暂停推送

#### 方法1: 禁用crontab

```bash
# 编辑crontab
crontab -e

# 在任务前添加 # 注释掉
# 0 9,13,20 * * * cd /root/project-wb/n8n && /usr/bin/python3 src/auto_push.py >> logs/auto_push.log 2>&1
```

#### 方法2: 禁用所有RSS源

```bash
# 使用jq工具批量修改（如果已安装）

mv /tmp/rss-temp.json /root/project-wb/n8n/config/rss-sources.json

# 或手动编辑，将所有 "enabled": true 改为 false
```

#### 恢复推送

```bash
# 方法1: 取消crontab注释
crontab -e
# 删除行首的 #

# 方法2: 重新启用RSS源
```

### 场景7: 导出本月推送统计报告

**操作步骤**:

```bash
# 生成本月统计报告
python3 << 'EOF'
import sqlite3
from datetime import datetime

db = sqlite3.connect('/root/project-wb/n8n/data/push_history.db')
cursor = db.cursor()

current_month = datetime.now().strftime('%Y-%m')

# 总推送量
cursor.execute('''
    SELECT COUNT(*) FROM push_history
    WHERE strftime('%Y-%m', pushed_at) = ?
''', (current_month,))
total = cursor.fetchone()[0]

# 各来源推送量
cursor.execute('''
    SELECT source, COUNT(*) as count
    FROM push_history
    WHERE strftime('%Y-%m', pushed_at) = ?
    GROUP BY source
    ORDER BY count DESC
''', (current_month,))

print(f"===== {current_month} 推送统计报告 =====\n")
print(f"总推送量: {total}篇\n")
print("各来源推送量:")
print("-" * 40)
for row in cursor.fetchall():
    source, count = row
    print(f"{source:20s}: {count:3d}篇")

db.close()
EOF
```
